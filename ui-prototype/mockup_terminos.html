Terminos de uso

Esta demo de SENDA URJC utiliza geolocalizacion durante el trayecto para ofrecer rutas seguras, avisos preventivos y funciones de emergencia.

El uso de la aplicacion implica aceptar las condiciones institucionales y el tratamiento de datos necesarios para las funciones de seguridad.

Politica de privacidad (resumen)

Se procesa ubicacion de forma temporal para monitorizacion de ruta y respuesta ante alertas SOS. Los datos se tratan con fines de seguridad del campus.

El usuario puede consultar, rectificar y solicitar informacion adicional segun la normativa vigente.

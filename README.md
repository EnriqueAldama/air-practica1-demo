# air-practica1-demo
## PASOS PARA VISUALIZAR LA DEMO DE LA APLICACIÓN MÓVIL
1. Clonar el repositorio con ` git clone https://github.com/EnriqueAldama/air-practica1-demo.git `

   O descargar el repositorio entero en zip y descomprimirlo

3. Abrir la carpeta `ui-prototype`

4. Localizar el archivo `index.html`

5. Abrir el archivo `index.html` en un buscador como Chrome

6. Interactuar con la demo
